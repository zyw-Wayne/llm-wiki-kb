# Skills 生态时间线

## 定义

Skills 从 Anthropic 在 Claude Code 中的功能到开放标准再到全行业生态的发展历程。

## 年度热词演进

| 年份 | 热词 | 含义 |
|------|------|------|
| 2023 | Prompt | 教 AI 说话 |
| 2024 | Agent | 教 AI 思考 |
| 2025 | MCP | 给 AI 开门 |
| 2026 | Skills | 给 AI 装技能 |

## 关键时间线

| 时间 | 事件 |
|------|------|
| 2025 年 10 月 16 日 | Anthropic 首次发布 Agent Skills，仅 Claude Code Pro 付费用户可用 |
| 2025 年 12 月 18 日 | **Anthropic 开放 Skills 标准**——类比安卓开源，Skills 从此属于所有 AI |
| 2026 年 1-2 月 | OpenAI（Codex）、Google、微软、Cursor、Antigravity、Trae、Qoder、CodeBuddy 等 Coding Agent 全面跟进 |
| 2026 年 2 月 | SkillsMP 市场收录 **16 万+ Skills**（三个月从零到十六万） |
| 2026 年春节后 | OpenClaw（龙虾）爆火、Hermes Agent（爱马仕）走热，均支持 Skills |
| 2026 年 Q1 | 桌面 Agent（Claude Cowork、Skywork、MiniMax Agent、扣子）陆续支持 Skills |

## 关键数据

- 三个月从零到 16 万，增速超过当年 APP Store 头三个月
- GitHub awesome-agent-skills 等汇总仓库 star 数动辄上万
- **Gartner 报告**：多智能体系统咨询量一年暴涨 **1445%**

## 行业判断

> "三年前我们教AI说话，两年前我们教AI思考，一年前我们给AI开门，今年我们给AI装技能。这条路的终点，是每个人都有一支按自己想法运转的AI团队。"
